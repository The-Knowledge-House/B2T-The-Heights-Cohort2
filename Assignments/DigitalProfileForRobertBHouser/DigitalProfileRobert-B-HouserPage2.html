He has extensive experience on project teams using the SDLC (System Development Life Cycle) methodology framework to complete enterprise projects and using Agile methodology framework for departmental applications.

Robert uses up-front planning and solution definition to provide general system documentation, statements of work, and full-page web applications using HTML5 and CSS.
